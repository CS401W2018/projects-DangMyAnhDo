Question:

1. Why is the viewport meta tag important?

It helps my site look good on different devices, especially phones.

2. How can you adjust the CSS for printing?

Use @media print to change how the page looks when printed.

3. When would you want to adjust the CSS for printing?

When we want to hide things, change the text size, or make it fit better on paper.
