Question:

1. Why do we need a viewport meta tag?

The viewport meta tag is needed so that the webpage looks good on mobile devices. Without it, the page could appear too small or zoomed out

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The last rule we write will take priority. So, if we write flex-direction: row; after flex-direction: column;, it will use row.

3. How do you change the CSS for a web page displayed on a phone?

We use media queries to make the page look good on smaller screens like phones.
