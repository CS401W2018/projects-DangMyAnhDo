Questions

1. What is normal flow in HTML?

Elements stack from top to bottom by default.

2. What does float do? Why do you need to clear the float?

It moves an element left or right, so things can wrap around it. We need to clear the float because clearing the float makes sure the container wraps around the floated element.

3. What do absolute, fixed, and relative do?

Absolute: positions an element based on its parent or the page.

Fixed: keeps the element in the same spot when we scroll.

Relative: moves an element from its original position

4. How can you make nice navigation buttons for mobile devices?
