Question 1: List out the different ways to select an item.

By ID: #id

By Class: .class

By Tag: tagname

By Attribute: attribute="value"

By Pseudo-classes: :hover, :focus

By Pseudo-elements: ::before, ::after.

Question 2: What is the difference between an ID and a Class?

ID used for one element and unique.

Class used for multiple elements.

Question 3: What is the difference between a SPAN tag and a DIV tag?

Span used for small section of the contents, it is inline element

DIV tag used for larger section, layout and block element.

Question 4: What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV is generic container.

HEADER contains the top section like a header.

FOOTER contains the bottom section like copyright.

Question 5: What are the different ways to select a color?

Name Color: Red, Blue, etc.

Hexadecimal: #ff5733

RGB: rgb(255,0,0)

RGBA: rgba(255,0,0.5)
