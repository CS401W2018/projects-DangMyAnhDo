Inline: it's when we write CSS directly in an HTML tag using the style attribute.

Embedded CSS: it's when we write CSS inside a style in the head section of the HTML.

External CSS: it's when we write CSS in a seperate file, which is a CSS file and link to our HTML.

We should use external CSS because it keeps our HTML clean and easy to manage, we can reuse the same file on different pages, it helps with faster page loading when we are working on it.

A selector is the way we pick which HTML element we want to style.
