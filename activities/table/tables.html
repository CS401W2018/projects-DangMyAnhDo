Question

1. Why don't we use the border attribute?

The border attribute is old and doesn't offer much control. It's better to use CSS because it gives us more options and is easier to work with.

2. What is the easiest way to color every other row a different color?

To color every other row differently, we can use CSS. Just apply a different color to even and odd rows using nth-child. For example, make even rows light gray and odd rows white.
