Question

1. When do you use a table?

I use tables when I need to display organized data in rows and columns, like lists, schedules, or comparisons.

2. When do you not use a table?

I avoid using tables for layout purposes—CSS is a better choice for that—or for simple lists, where a ul or ol would be more appropriate.

3. When do you need to do extra work for table accessibility?

I use th for headers, add a caption to explain the table, and use the scope attribute to help screen readers. This makes tables easier to use
