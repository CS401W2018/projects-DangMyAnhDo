Question 2: Explain the difference between an absolute link and a relative link. Why would you use relative links?

Abcolute Link: other domains and the path to the resource, start with "https://....", this link always points to the same resource, no matter where the link is used on the web

Relative Link: within your own domains, not technically required, this link works based on where the file is located on the website, should use this

Question 4: List the 3 rules with relative links.

  1. Same folder: the file is in the same folder as our page, just use the same file name.
  2. Subfolder: the file is in a folder inside our current folder, include the folder name.
  3. Go Up a folder: the file is in a folder above our current one, use ../to go up to one level.
