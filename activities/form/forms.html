Question:

1. What is JavaScript? How did it get its name?

Website code that makes things interactive. It was named JavaScript to capitalize on the popularity of Java, even though they are distinct programming languages.

2. What are the two ways you can run JavaScript?

In a web browser, or on a server.

3. What was the most popular front-end js library?

jQuery

4. What is the most popular front-end js library today?

React (also Vue and Angular).
