Internet

The internet is a big network that connects computers. It helps people share information, send messages, and do things like watch videos or browse websites.

Web

The web is a part of the internet where we can visit websites and look at information, videos, and pictures. It is what we use when we open a browser.

How are the Internet and Web different?

The internet is the network that connects computers around the World and the web is one part of it, where we visit websites and look for information

ASCII and Unicode

ASCII it is a system with 128 characters such as letters, numbers and symbols to represent text but it's limited and mostly used for English.

Unicode it is a system that can represent text from almost every language, with over 143,000 characters.

What is UTF-8?

UTF-8 is a way to store text that can handle any character or any language. It uses 1 to 4 units to store each character.

URL

A URL (Uniform Resource Locator) is the address used to access a specific account, such as a website or the internet. It specifies the location and method used to retrieve it.

Domain Name

A domain name is the name of the website on the internet, like "google.com". It makes it easier for users to find website without having to memorize complex IP addresses.

HTTP

HTTP (HyperText Transfer Protocol) is a system that helps tranfer data, like web pages, from a server to browser and we can see them. It is the foundation of any communication on the web and is also the protocol between clients and servers.

HTTPS

HTTPS (HyperText Transfer Protocol Secure) is a secure hypertext transmission protocol, it keeps users information secure when the user visit a site.

IP Address

An IP address is a unique number assigned to each device on the internet, like a phone or computer. It facilitates device communication and discovery with each other.

DNS

DNS (Domain Name System) is a system name resolution domain, which easily convert name domains into IP addresses. This is a necessary process for the browser to access the website.

W3C Recommendations

W3C Recommendations are guidelines and standards set forth by World Wide Web Consortium (W3C) to help web technologies ensure consistent use and best performance across different devices and platforms. Standards that make websites accessible, usable, trustworthy, and secure.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is a way to store and organize data in a structured format and it helps us to store information but does not define how it looks.

XHTML (eXtensible HyperText Markup Language) is a more stringent variation of HTML. It combines HTML with XML rules to improve structure and help with error handling.

HTML (HyperText Markup Language) is the basic language used to create and structure web pages. Determine the content and layout of a website.

HTML5 represents the most recent iteration of HTML. It improves new functionality, features, and enhancements, making it easy to modernize websites with better support for rich meadia, activities, and interactive content.
